
# R Wallet – Powered by Rapicoin

This repository contains the official Android and Windows wallets for Rapicoin (R Wallet).

## 🔹 Android Wallet
- Filename: `rapicoin.apk`
- Function: Send, receive and store Rapicoin securely

## 🔹 Windows Wallet
- Filename: `rapicoin-qt.exe`
- Function: Full desktop wallet with all Rapicoin blockchain functionalities

## 🛠 Features
- Send/receive Rapicoin
- Backup/import/export wallet
- Transaction history
- QR code for address sharing

## Note
Android wallet does not support mining. Mining requires a PC with Rapicoin mining software.
